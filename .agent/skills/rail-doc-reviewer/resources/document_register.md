### 1. ICD027_C — Ormeau Signalling (Conventional – ETCS)

| Field | Value |
|---|---|
| Document Number | QTMP-AAA-DOW-EM-ICD-000027 |
| Revision | C (Draft) |
| Status | IFU (Issued For Use) |
| Classification | ICD |
| Interface Parties | Downer Depot (OMC) ↔ QR Stabling Yard (ETCS within SY / Sequence Alliance) |

**Executive Summary**: Defines the signalling interface between the Ormeau Maintenance Facility (conventional) and the Ormeau Stabling Yard (ETCS). Staged approach: **Stage 1** commissions the Maintenance Facility in isolation with physical safeguards (stop boards, clipped/locked points, no data exchange). **Stage 2** enables full ETCS integration with signalling slotting, interlocking data exchange, and coordinated entry/exit movements via relay-to-relay interface.

**Key Findings**:
- Stage 1: Southern connection operational with verbal handover between Downer YM and QR RMC. Northern connection non-operational (points clipped/locked).
- Stage 2: Slot request/release mechanism for all entry/exit movements. One-shot slot requests only.
- Physical interface via 2× 50-core cables between MF06 SER and OMS2 location case.
- 4 relay interface tables defined (Southern in/out, Northern in/out) with vital and non-vital outputs.
- Interface Location Case powered from Stabling Yard signalling system at 110V.

**Interface Dependencies**: ICD045 (ETCS signalling scope), ICD032 (telecoms), ICD046 (mainline connection), Schedule C1 (scope split matrix).

**Open Actions / Risks**:
- Document not yet endorsed (no signatures on endorsement page).
- Baseline documents are "work-in-progress" and not yet approved.
- Test & Commissioning section is a placeholder — details deferred to respective parties' T&C plans.
- Table 4 (Northern connection) references show relay naming inconsistencies (e.g., OMC.81B.T1PR described as "12AT Track Repeat").

**Role Relevance**: **Critical** — directly relevant to ETCS/RCS interface management, commissioning strategy (staged approach), and RIM handover planning.
